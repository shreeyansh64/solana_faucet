# Solana Devnet Faucet (React)

A simple and fast faucet for requesting SOL on **Solana Devnet**, built with React and `@solana/web3.js`.

## Features
- Request SOL on Devnet  
- Clean, responsive UI  
- Transaction status + error handling  

## Setup
```bash
git clone https://github.com/shreeyansh64/solana_faucet.git
cd Sol_Airdrop
npm install
npm run dev
```
